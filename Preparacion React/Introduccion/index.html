Javascript desde cero

¿Qué es Javascript?

Es un lenguaje de propgramación, interpretado, orientado a objetos, imperativo y de tipado débil; que tiene como funcón darle dinamismo a una página web. También se puede usar para inteligencia artificial, robótica y aplicaciones de escritorio.

¿Cómo se enlaza javascript con HTML?

  1. En linea: Poniendo el código como valor de un atributo en un elemento html

    2. <elemento onclick = "alert('Mensaje')"> </elemento>

  2. Etiqueta scipt: Se hace uso de la etiqueta script, depositando el código de javascript como su contenido.

    3. <script>
    alert("Enlace por etiqueta")
    </script>

  3. Archivo js: Se realiza poniendo la ruta del archivo como valor del atriobuto src de la etiqueta script

    4. <script src="ruta al script"> </script>
