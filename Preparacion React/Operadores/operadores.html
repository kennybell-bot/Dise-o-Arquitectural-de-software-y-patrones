Operadores JavaScript

Los operadores son símbolos que sirven para indicar que hacer con uno o mas datos. Dependiendo de la cantidad de datos que estos requieren para funcionar, se pueden encontrar de tipo binario(dos datos), unarios (un dato) y ternarios (tres datos). También estos se clasfican deacuerdo a su función de la siguiente manera:

Asignación

Son los operadores que sirven para asignarle un valor a una determinada variable, los operadores que se encuentran dentro de este grupo son los siguientes:

  • =: Sirve para asignarle un valor a una variable.

    let x=y

  • +=: Sirve adicionar un valor a la variable y posteriormente guardar dicho resultado en la misma variable.

    let x+=y

  • -=: Realiza lo mismo que el operador anterior, pero esta resta en vez de sumar.

    let x-=y

  • *=: este operador multiplica la variable por un número y aloja el resultado en la misma variable.

    let x*=y

  • /=: Realiza lo mismo que los operadores anteriores pero este divide el por el valor que se le indique.

    /=: Realiza lo mismo que los operadores anteriores pero este divide el por el valor que se le indique.

  • %=: Este operador divide pero consigna en la variable el residuo de dicha división.

    let x%=y

  • **=: Este operador aloja en la variable el resultado de elevarla al número que se le indique.

    let x**=y

  • <<=: Opera a nivel binario, lo que hace es asignarle a la variable el valor resultante de mover un bit a la derecha la representación binaria de lo que había alojado antes.

    let x<<=y

  • >>=: Hace lo mismo pero esta vez el desplazamiento de bits es hacia la izquierda.

    let x>>=y

  • >>>=: este operador hace los mismo que los anteriores, sin embargo este desplaza los bits un número específicado de posiciones y además no conserva el bit del signo.

    let x>>>=y

  • &=: este operador indica que en la variable se alojara el resultado de aplicar una operación AND bit por bit.

    let x&=y

  • ^=: Este es un XOR bit a bit.

    let x^=y

  • |=: OR bit a bit.

    let x|=y

  • &&=: Esta es la asignación de un AND lógico

    let x&&=y

  • ??=: Este operador sirve para asignarle un valor a la variable únicamente si esta tiene como valor null.

    let x??=y

  • Desestructuración: Es una forma de asignarle valores de un arreglo o un objeto a un conjunto de variables, tiene una nortación similar a los arreglos.

    var foo = ["one", "two", "three"];
    var [one, two, three] = foo;

Comparación

Estos son operadores que permiten realizar la comparación entre dos datos y devuelven un valor booleano (true o false) dependiendo del valor de los datos. Estos dentro de su labor, realizan una conversión de los datos, en el caso de que estos sean de tipos diferentes, para poder retornar un resultado de la operación.

  • ==: Sirve para comparar si dos datos son iguales.

    x==y

  • !=: Sirve para establecer si dos datos son diferentes.

    x!=y

  • >: Sirve para averiguar si un valor es mayor que otro.

    x>y

  • >=: Sirve para averiguar si un valor es mayor o igual que otro.

    x>=y

  • <: Sirve para establecer si un valor es menor que otro.

    x<y

  • <=: Sirve para establecer si un valor es menor o igual que otro

    x<=y

Sin embargo, hay ciertos operadores que no realizan esta conversión y son los siguientes:

  • ===: Sirve para comparar si dos datos son estrictamente iguales.

    x===y

  • !==: Sirve para establecer si dos datos son estrictamente diferentes.

    x!==y

Aritméticos

Son los operadores que permiten hacer las conocidas operaciones entre números (suma, resta, multiplicación y división) y otras operaciones menos conocidas

  • +: Sirve para sumar dos números.

    x+y

  • -: Sirve pra hacer la resta de dos números.

    x-y

  • *: Sirve para hacer la multiplicación de dos números

    x*y

  • /: sirve para hacer la división de dos números.

    x/y

  • %: LLamada módulo, da como resultado el residuo de la división entre dos números.

    x%y

  • ++: Sirve para incrementar en uno el valor de un número

    x++

  • **: Sirve para elevvar un número a una determinada potencia.

    x**y

  • --: Decrementa en uno el valor de un número.

    x--

  • +: Sirve para volver positivo un número

    +x

  • -: Sirve para volver negativo un número

    -x

Bit a bit

Estos son los operadores que trabajan con los valores en binario.

  • &: reliza la operación AND bit por bit.

    x&y

  • |: Realiza una operación OR bit por bit.

    x|y

  • ^: Realiza la operación XOR bit por bit.

    x^y

  • ~: Realiza la negacionbit por bit

    ~x

  • >: Realiza el desplazamiendo de una determinaada cantidad de bits a la derecha del valor binario

    x>>y

  • <<: Realiza le corrimiento pero a la izquierda

    x<<y

  • >>>: Realiza el desplazamiendo de los bits sin conservar el bit del signo

    x>>>y

Lógicos

Estos operadores manejan las operaciones logicas con valore booleanos

  • &&: Realiza la operación AND lógica

    x&&y

  • ||: Operación OR lógica

    x||y

  • !: Negación lógica, operación NOT

    !x

Cadenas

Los operadores de cadenas, son principalmente para la concatenación (unir dos cadenas).

  • +: Concatena dos cadenas

    x+y

  • +=: Otra forma de concatenar cadenas

    x+=y

Condicional ternario

Este condicional tiene tres partes:

  • Condición.

  • Valor 1: Que es el que retornará si la condición es verdadera.

  • Valor 2: Se retorna si la condición es falsa.

condición ? valor1 : valor2
