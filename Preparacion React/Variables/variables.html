Variables Javascript

¿Qué es una variable?

Una variable es un espacio en memoria en el que se pueden almacenar datos para ser recuperados en cualquier momento que sea necesario. Además estos valores pueden ser utilizados en diferentes operaciones. En Javascript, el proceso para indicar la creación de una variable empieza por su declaración y finaliza con la asignación del valor que almacenará.

Declaración

En el proceso de declaracó hay que tener en cuenta que alcance se ncesita para la variable, es decir si esta solo se tendrá en cuenta en el bloque de código en el que se declaró o se tendrá en cuenta también fuera del mismo. Para este fin, se tienen las palabras clave:

  • let: De alcance local.
  • var: De alcance global.

Siguiente a la plabra clave que define el alcance de la variable, se pone el nombre con el que se hará referencia a la misma, hay que tener en cuenta reglas generales para el establecimieto de esta como:

  1. El nombre de la variable no debe empezar con un número o caracter esepecial.
  2. Los nombres de las variables deben ser descriptivos de su contenido y fin.
  3. JavaScript diferencia entre mayúsculas y minúscula, es decir hola y Hola son variables diferentes.

Finalmente, después de la declaración y el nombre se cierra con ; la línea para indicar que allí finaliza una instrucción. Sep ueden declarar varias variables en una sola línea separandolas por comas.

let variableLocal;
var variableGlobal;
let variable1, variable2;

Adicionalmente, en JavaScript se puede inicializar una variable con la palabra const, esto para indicar que será constante y su valor no cambiará una ves le sea asignado por primera vez.

const variableConstante;

Asignación

La asignación se refiera a la acción de indicar qué valor se alojará en la variable. Esto se logra mediante el uso del signo igual (=), seguido del valor indicado. Una varible puede se le puede aignar un valor en la misma línea en la que se declara.

let variableLocal = 1;
var variableGlobal = “Hola”;

Tipos de dato

Los valores que se pueden guardar en una variable pueden ser de diferente naturaleza.

  • Números: Dentro de las variables se pueden consignar números, estos pueden ser enteros como 30 o 15. También se pueden guardar números decimales
  • Cadenas: Son el tipo de datos que s utiliza para guradar texto en la variable, estas se indican usando comilas “”.
  • Booleanos: Los booleanos, son tipos de datos de solo dos valores posibles, true (verdadero) o false (falso), estos son útiles en estructuras de control.
  • Arreglos: Los arreglos son elementos que pueden almacenar un conjunto de valores, se notan con el uso de [] y los valores en su interior se separan con coma.
  • Objetos: Son elementos que son representaciones de objetos de la vida real en código, estos se notan con {} y en su interior, se encuentran sus atributos y métodos.

Tipado dinámico

Es una de las caraterísticas de JavaScript y de varios lenguajes de programación, de poder almacenar valores de diferentes tipos a lo largo de la ejecución del programa. Además, esto implica que al declarar la variable no sea necesario indicar el tipo de dato de los valores que va a almacenar. Con la función typeof() se puede obtener el tipo de dato del valor que tiene alojado la varible en ese momento.

let variable = “cadena”;
typeof(variable);
