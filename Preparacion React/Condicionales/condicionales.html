Condicionales

Los condicionales son estructuras que nos permiten elegir que fragmento de código ejecutar, tomando como referencia el valor de verdad (booleano) de una condición, construida con el uso de operadores lógicos y/o de comparación.

Estructura if(){}...else{}

Esta estructura se compone de la palabra clave “if” seguida de la condición que se desea establecer para la ejecución del código, esta se coloca dentro de parentesis. Seguido a ello va el fragmento de código que se ejecuta si la condición es verdadera, este se coloca entre llaves. Posteriormente se encuentra la palabra clave else, seguida del código que se ejecuta si la condición definida es falsa, este último código también va entre llaves.

if(texto==”if”){
alert(“Se ejecutó el primer fragmento de código”);
}else{
alert(“Se ejecutó el último fragmento de código”);
}

else if()

Esta expresión permite encadenar varias condiciones antes de cerrar el condicional. Estas expresiones se colocan una a continuación de la otra, las veces que sea necesario y con las condiciones que requiera cada una en sus respectivos parentesis.

if(texto==”if”){
alert(“Se ejecutó el primer fragmento de código”);
}else if(texto==”elseif”){
alert(“Se ejecutó el código del else if”);
}else{
alert(“Se ejecutó el último fragmento de código”);
}

Anidación

La anidacion de los condicionales consiste en poner un condicional en el fragmento de código de otro

if(texto==”if”){
if(texto==”if”){
alert(“Se ejecutó el condicional anidado”);
}
}else{
alert(“Se ejecutó el último fragmento de código”);
}

Switch

Es una estructura condicional, esta se compone de la palabra clave switch seguida de la variable o expresion que se desea comparar entre parentesis. Posteriormente, entre llaves, se utiliza la plabra clave case seguida del valor con el que queremos comparar la expresion o variable del principio, despues de dosp untos (:), en una linea nueva y con identación, se pone el código que se ejecutará si la expresión coincide con el valor del case. Para finalizar se pone la línea “break;” para indicar que ya se puede salir de la estructura condicional sin necesidad de revisar las otras estructuras case. Como cierre de la estructura switch, se tiene la palabra clave “default”, que es la que contiene el código que se ejecuta si la variable o expresión no coincide con ninguno de los valores definidos en las estructuras case.

switch(variable){
case “1”;
alert(“Se ejecutó el primer case”); break;
case “2”:
alert(“Se ejecutó el segundo case”); break;
default;
alert(“Se ejecutó el default”)
}

Operador ternario

Esta esctructura permite retornar un valor si la condición establecida es true y otro si es false. Esta se compone de la condición, seguida del signo de interrogación, posteriormente se diligencia el valor que se retorna si la condición es verdadera, a continuación se colocan dos puntos (:) y finalmente se coloca el valor que se obtendra si l;a copndicion es falsa.

variable == “ternario” ? alert(“Es verdad”) : alert(“es falso”);
