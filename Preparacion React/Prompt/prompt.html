Función promt

Es una función que crea una ventana modal con un campo de texto de una línea para que el usuario ingrese un texto. La función retornara dicho texto como una cadena que podrá ser guardada en una variable. La ventana emergente cuenta con dos opciones, Ok que enviara el texto que el usuario ingresó, o el valor por defecto en caso de que no lo haya hecho, y Cancel que hará que se retorne un valor null (vacío).

Parámetros

Esta función cuenta con dos parámetros que sirven para manipular el mensaje que se va a mostrar en la ventana y el valor por defecto que llevará la cadena cuando el usuario no ingrese nada

let palabra = prompt(mensaje, valorPorDefecto)
