Concatenación

La concatenación es la operación de unir dos o mas cadenas. Esta se lleva a cabo per medio del uso del operador +, del método de las cadenas .concat() o del uso de la expresión ${}.

Operador +

El uso de este operador consiste en ponerlo entre las cadenas que se deseen concatenar.

let cadenaUno = “Hola”;
var variableDos = “Muundo”;
lalert(cadenaUno + variableDos)

Método concat()

El método concat es propio de las cadenas. Sirve para concatenar una cadena con otro valor que puede ser un número.

'cadena'.concat(5);

Expresión S{}

El uso de esta expresión consiste en ponerla dentro de la cadena en la posición deseada, con el valor que se necesita incluir entre llaves. Además, la cadena debe estar entre backtits (` `), de lo contrario esta forma de concatenar no funcionará.

`Este es un ${"ejemplo"}`

Escape de comillas

El escape de comillas consiste en que en para poder mostrar comillas simples o comillas dobles a la hora de mostrar el contenido de la cadena, se debe encerrar la cadena con un tipo de comillas diferentes a las que se quiere mostrar en la cadena.

let cadena = “Hola ‘mundo’”
